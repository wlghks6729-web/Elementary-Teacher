두 자리 ÷ 한 자리 챗봇형 튜터 - 사용 안내

  1. GitHub 에 새 저장소를 만듭니다(예: division-tutor).
  2. 이 폴더의 파일(index.html, README.html)을 저장소 루트에 업로드합니다.
  3. 저장소에서 Settings → Pages 로 이동해서, Deploy from a branch, Branch를 main과 / (root)로 설정 후 저장합니다.
  4. 잠시 후 생성된 주소(예: https://사용자명.github.io/division-tutor/)를 학생들과 공유하세요.

Tailwind/React CDN을 사용하므로 추가 빌드 과정 없이 바로 동작합니다.
